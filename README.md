# Zoo Animal Information Card
Thisproject is a single-page website created for a web development lecture.

The webpage presents information about insects from the class *Insecta*.
It is designed like a zoo exhibit card showing details about the animal group.
## Features
-HTML page structure
-Navigation bar
-Styled informaion card
-animal details table
-order.unorder and definition lists
-CSS design with box shadow and etc.
