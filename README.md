## Zoo Animal Information Card
This is a simple one-page that presents information about insects in a zoo exhibit style. It is designed for a web development class project.

## Project Discription
The webpage presents information about insects from the class Insecta. It is designed like a zoo exhibit card showing details about the animal group.

## Technologies used
   -HTML
   -CSS
## Features
  -Navigation menu
  -Image display 
  -rounded card design
